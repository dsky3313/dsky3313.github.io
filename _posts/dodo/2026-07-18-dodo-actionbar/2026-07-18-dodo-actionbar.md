## ■ 아이콘 색상

아이콘 상태에 따라 탈색·색상 변경을 적용합니다.

<!-- 이미지: 색상 비교 (기본 vs dodo) -->

| 상태 | 효과 |
|------|------|
| 사거리 밖 | 색상 변경 + 탈색 |
| 마나 부족 | 색상 변경 + 탈색 |
| 사용 불가 | 탈색 |
| 쿨다운 중 | 탈색 |

기본 WoW는 사거리 밖이면 빨간 테두리, 사용 불가면 어둡게 처리하는 방식인데, dodo는 아이콘 자체를 탈색 + 색조 변경해서 상태를 구분합니다.

## ■ 단축키 / 매크로 텍스트

단축키 텍스트, 매크로 이름을 바별로 숨길 수 있습니다.

<!-- 이미지: 텍스트 숨김 비교 -->

단축키를 숨겨도 **사거리 인디케이터(●)** 는 그대로 표시됩니다.

| 옵션 | 설명 |
|------|------|
| 텍스트: 단축키 | 버튼 단축키 숨기기 |
| 텍스트: 매크로 | 버튼 매크로 이름 숨기기 |

## ■ 차단 오버레이

타겟 또는 포커스가 스킬을 시전할 때, 차단 스킬 버튼에 오버레이를 표시합니다.

<!-- 이미지: 차단 오버레이 -->

포커스가 있으면 포커스 우선으로 감시합니다.

- 차단 가능: 초록 애니메이션 + 잔여 시전 시간
- 차단 불가 (면역 등): 흐리게 표시
- 쿨다운 중: 색상 구분 (빨강→노랑→흰색)

지원 스킬 목록:

| 직업 | 스킬 |
|------|------|
| 죽음의 기사 | 정신 동결 |
| 악마 사냥꾼 | 분열 |
| 드루이드 | 태양 빛 차단, 두개골 강타 |
| 사냥꾼 | 대응 사격, 총구 소음기 |
| 마법사 | 반마법 |
| 수도사 | 창수리 공격 |
| 성기사 | 꾸짖음 |
| 사제 | 침묵 |
| 도적 | 발차기 |
| 주술사 | 바람 절단 |
| 흑마법사 | 주문 봉쇄, 도끼 던지기 |
| 전사 | 강타, 분쇄 |

## ■ 강화효과 오버레이

버프가 걸린 스킬 버튼에 남은 시간·스택을 오버레이로 표시합니다.

<!-- 이미지: CDM 오버레이 -->

WoW 기본 버프바 쿨다운 뷰어를 후킹해서 버프를 감지합니다.

- 남은 시간: 버튼 좌상단 **녹색** 텍스트
- 스택 수: 버튼 우하단 표시
- 초록 내부 광채 오버레이

일부 직업 특화 매핑이 포함돼 있습니다.

| 버프 | 연결 스킬 |
|------|----------|
| 집행자의 정밀함 | 필사의 일격 (무기 전사) |
| 격노 | 광란 (분노 전사) |
| 소용돌이 연마 | 소용돌이 (분노 전사) |

물약처럼 CDM API를 지원하지 않는 아이템은 사용 이벤트를 직접 감지해서 처리합니다.

---

## ■ 물약 오버레이

전투 중, 물약이 쿨다운 없이 가방에 있을 때 버튼에 빛나는 오버레이를 표시합니다.

<!-- 이미지: 물약 오버레이 -->

전투 밖에서는 표시되지 않습니다.

---

## ■ 설정 방법

Edit Mode (`Ctrl+Shift+?` 또는 `/editmode`) 에서 각 행동단축바를 선택하면 dodo 설정 항목이 표시됩니다.

<!-- 이미지: Edit Mode 설정 패널 -->

모든 기능은 바별로 개별 설정 가능합니다.

모듈 마스터 토글은 Edit Mode 패널 좌측 **dodo 모듈 → 전투 → 행동단축바** 에서 켜고 끌 수 있습니다.
